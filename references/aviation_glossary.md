# Aviation MRO/CAMO Glossary

## Document Types

| Term | Full Name | Description |
|------|-----------|-------------|
| AMM | Aircraft Maintenance Manual | OEM document with all maintenance tasks |
| AMP | Approved Maintenance Programme | Airline's EASA-approved scheduled maintenance plan |
| MPD | Maintenance Planning Document | OEM's scheduling guidance document |
| CMR | Certification Maintenance Requirement | Mandatory scheduled task (failure would affect airworthiness) |
| ALI | Airworthiness Limitation Item | Hard-limit task, FAA/EASA approved, cannot be deviated |
| SB | Service Bulletin | OEM-issued modification or improvement (may become mandatory via AD) |
| AD | Airworthiness Directive | Mandatory safety requirement issued by authority |
| TR | Temporary Revision | Urgent AMM update before next normal revision |
| LEP | List of Effective Pages | Index showing revision status of every AMM page |
| SRM | Structural Repair Manual | Structural damage limits and repair procedures |
| CMM | Component Maintenance Manual | Off-aircraft component maintenance instructions |
| IPC | Illustrated Parts Catalog | Parts identification and ordering reference |

## Organization Types

| Term | Description |
|------|-------------|
| MRO | Maintenance, Repair & Overhaul organization (Part 145) |
| CAMO | Continuing Airworthiness Management Organization (Part M/CAMO) |
| OEM | Original Equipment Manufacturer (Boeing, Airbus, Cessna, etc.) |
| Part 145 | EASA approval for maintenance organizations |
| Part M | EASA regulation for continuing airworthiness management |
| Part CAMO | EASA approval specifically for CAMO organizations |

## Technical Terms

| Term | Description |
|------|-------------|
| FH | Flight Hours |
| FC | Flight Cycles (one takeoff + landing) |
| MH | Man-Hours (labor estimate) |
| ATA | Air Transport Association (chapter numbering standard) |
| iSpec 2200 | Modern ATA document specification standard |
| MSN | Manufacturer Serial Number |
| FSN | Fleet Serial Number (operator's internal reference) |
| NDT | Non-Destructive Testing (includes UT, ET, PT, MT, RT) |
| UT | Ultrasonic Testing |
| ET | Eddy Current Testing |
| PT | Dye Penetrant Testing |
| MT | Magnetic Particle Testing |
| RT | Radiographic Testing (X-ray) |
| GVI | General Visual Inspection |
| DI | Detailed Inspection |
| SDI | Special Detailed Inspection (with NDT) |
| BITE | Built-In Test Equipment |
| MEL | Minimum Equipment List |
| MMEL | Master Minimum Equipment List |

## Interval Types

| Type | How Measured |
|------|-------------|
| Calendar | Days, months, years — elapsed regardless of operation |
| Flight Hours (FH) | Airborne time (wheels up to wheels down) |
| Flight Cycles (FC) | One landing cycle |
| Engine Hours | Engine-on time (may differ from FH) |
| APU Hours | Auxiliary Power Unit runtime |

## Priority Levels for AMM Changes

| Level | Trigger | Action Required |
|-------|---------|----------------|
| MANDATORY | Interval reduced, new CMR/ALI | AMP amendment, authority notification |
| HIGH | New mandatory task, deleted task | Engineering evaluation within 30 days |
| MEDIUM | New recommended task, tool/material change | Plan for next opportunity |
| LOW | Clarification, editorial change | Note in records |
